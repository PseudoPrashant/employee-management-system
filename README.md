# Employee Management System (PHP + MySQL)

A web-based **Employee Management System** built using **Core PHP** and **MySQL**. This project allows an admin to manage employee records through a simple dashboard with authentication, CRUD operations, and search/filter functionality.

---

## Features

- Admin Login & Logout (Session-based Authentication)
- Add New Employee
- View Employee List
- Edit Employee Details
- Delete Employee Record
- Search Employees (by Name / Email)
- Filter Employees by Department
- Clean UI with HTML & CSS
- Optimized MySQL Queries

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP (Core PHP)  
- **Database:** MySQL  
- **Tools:** XAMPP, phpMyAdmin, VS Code
